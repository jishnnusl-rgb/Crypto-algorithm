📚 Mode of Operations

ECB (Electronic Codebook): Each block is encrypted independently. Simple but patterns in plaintext may be visible in ciphertext.


CBC (Cipher Block Chaining): Each block is XORed with the previous ciphertext block before encryption. More secure as identical plaintext blocks produce different ciphertext.
