ℹ️ About CMAC (Message-Only Mode)

Complete Flow:

1️⃣ Input: Message only (no separate key input)

2️⃣ MD5 Hash: Message → 128-bit MD5 hash

3️⃣ Key Extraction: First 64 bits of MD5 → DES key

4️⃣ Subkey Generation: DES(0⁶⁴) → K1 & K2

5️⃣ Message Padding: Add 0x80 + 0x00 if needed

6️⃣ Block Processing: XOR blocks, apply subkey to last, DES encrypt

7️⃣ Output: n-bit CMAC (truncate if needed)

Subkey Selection:

• K1: Complete block (no padding)

• K2: Incomplete block (padding needed)

Security: Provides message authentication & integrity
