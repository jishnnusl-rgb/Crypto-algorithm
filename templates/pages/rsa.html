RSA Algorithm Overview:

Key Generation: Choose primes p, q → n = p×q → φ(n) = (p-1)(q-1) → Choose e → d = e⁻¹ mod φ(n)

Encryption: C = Me mod n

Decryption: M = Cd mod n

📐 RSA Formulas

Key Generation
n = p × q φ(n) = (p-1)(q-1) d = e⁻¹ mod φ(n)
Encryption
C = Me mod n

M = plaintext, C = ciphertext

Decryption
M = Cd mod n

Using private key d

Requirements
gcd(e, φ(n)) = 1 1 < e < φ(n)

e and d are inverses

💡 Example

Step 1: Choose Primes

p = 61, q = 53

n = 61 × 53 = 3233

Step 2: Calculate φ(n)

φ(n) = (61-1)(53-1)

φ(n) = 60 × 52 = 3120

Step 3: Choose e

e = 17

gcd(17, 3120) = 1 ✓

Step 4: Calculate d

d = 17⁻¹ mod 3120

d = 2753

Encrypt 'A' (65)

C = 6517 mod 3233

C = 2790

Decrypt 2790

M = 27902753 mod 3233

M = 65 = 'A'
