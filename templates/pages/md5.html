ℹ️ About MD5

MD5 (Message Digest Algorithm 5) is a widely used cryptographic hash function.

Characteristics:

• Produces 128-bit (16-byte) hash value

• Processes message in 512-bit (64-byte) blocks

• Uses 4 rounds with 16 operations each (64 total)

• Four auxiliary functions: F, G, H, I

• Note: MD5 is cryptographically broken - use for learning only!

MD5 Rounds:

• Round 1 (0-15): F(B,C,D) = (B ∧ C) ∨ (¬B ∧ D)

• Round 2 (16-31): G(B,C,D) = (B ∧ D) ∨ (C ∧ ¬D)

• Round 3 (32-47): H(B,C,D) = B ⊕ C ⊕ D

• Round 4 (48-63): I(B,C,D) = C ⊕ (B ∨ ¬D)
